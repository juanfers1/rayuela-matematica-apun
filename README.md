# Rayuela para entender el mundo

Ensayo interactivo estático para la Cátedra APUN. Relaciona la sesión **Las matemáticas: “una ventana para entender el mundo que habitamos”**, de Juan Carlos Maya López, con *Rayuela* de Julio Cortázar y *María* de Jorge Isaacs.

## Publicación en GitHub Pages

El proyecto incluye un flujo automático en `.github/workflows/pages.yml`. Al subirlo a un repositorio y activar **Settings → Pages → Source: GitHub Actions**, cada cambio en la rama `main` genera y publica la carpeta estática.

También puede generarse manualmente con `npm run build:pages`. El resultado queda en `gh-pages/` y usa rutas relativas, por lo que funciona dentro de cualquier subcarpeta de GitHub Pages.

## Recorridos

- Ruta salto: parte de la estructura no lineal de *Rayuela*.
- Ruta paisaje: parte del territorio sensible de *María*.
- Ruta aleatoria: reorganiza las seis estaciones centrales.
